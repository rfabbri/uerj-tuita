These files are to be used with Oysttyer

1) generate your ~/.oysttyerrc with your APP credentials, per
http://oysttyer.github.io. For the UERJ effort, request this file from user:
github.com/rfabbri

2) login into your desired username on the browser

3) run oysttyer

4) Authorize the app per oysttyer's wizard instructions, and type in the code

5) run oysttyer and type to tweet

6) quit, then run randomtweet: starting this app will run an infinite loop,
selecting random lines from the file tweets.txt

7) optionally, put on your crontab so that it starts by itself
